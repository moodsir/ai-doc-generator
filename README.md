# ai-doc-generator
一个基于 AI 的自动化项目文档生成工具，可快速为项目生成结构化的技能文档（Skill）。
An AI-powered automated project documentation generator that quickly creates structured skill documents for your projects.

## ✨ 功能特性 | Features
- 自动解析项目结构，生成标准化技能文档
  Auto-parses project structure and generates standardized skill documents
- 支持 Python、HTML 等多语言项目
  Supports multi-language projects including Python and HTML
- 一键生成可用于 AI 智能体的 Skill 描述文件
  One-click generation of Skill description files for AI agents

## 📁 项目结构 | Project Structure

ai-doc-generator/
├── .trae/skills/ai-doc-generator/ # 自动生成的技能文档目录 | Auto-generated skill docs directory
├── .gitignore # Git 忽略文件配置 | Git ignore file configuration
├── app.py # 核心业务逻辑代码 | Core business logic code
└── index.html # 前端展示页面 | Frontend display page

## 🚀 使用方法 | Usage
1.  将项目代码放入对应目录
    Place your project code in the corresponding directory
2.  运行 `python app.py` 启动文档生成服务
    Run `python app.py` to start the documentation generation service
3.  访问 `index.html` 查看生成的文档效果
    Visit `index.html` to view the generated documentation
